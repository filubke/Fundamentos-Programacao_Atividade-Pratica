Bem-vindo ao Meu Portfólio

Aqui veremos alguns trabalhos que fiz anteriormente.

Um dos trabalhos mais interessantes que realizei foi um web scraping em uma página com dados do Brasileirão de 2024 para o curso de estatística na UFPR.
O objetivo era coletar esses dados e, em seguida, realizar uma análise. Escolhi examinar a diferença de desempenho entre jogos em casa e fora, tentando determinar se esse fator influenciava a posição da equipe no campeonato.
Além da análise em Python, utilizando pacotes para coleta dos dados, geração de gráficos e insights, também foi gerado um arquivo Markdown para leitura posterior, além do clássico PDF. No entanto, o Markdown apresenta vantagens por ser responsivo.

Apesar de ter visto que é possível carregar o arquivo Markdown (.md) no html, não irei subir o arquivo devido a limitações de conhecimento, mas o PDF está disponível no seguinte LINK.
